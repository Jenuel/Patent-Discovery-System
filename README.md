# Patent Discovery System

Welcome to the Patent Discovery System repository! This project is an AI-powered patent search and analysis platform, designed to assist users in uncovering prior art, performing infringement/clearance tasks, and exploring patent landscapes.

The solution is divided into a frontend web application and a backend API server, supported heavily by advanced AI and search models (Google Gemini, OpenAI embeddings, Elasticsearch, Pinecone, and MongoDB).
